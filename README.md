# Resources for hacking on Open Web Apps
## Web App Templates
* [mortar app stub](https://github.com/mozilla/mortar-app-stub): for any app; a blank template that comes only with the basic things required to get started | [Download](/mortar-app-stub-0.1.0.zip)
* [mortar game stub](https://github.com/mozilla/mortar-game-stub): for building 2D games with canvas; a minimal web game template that handles input and a render loop | [Download](/mortar-game-stub.0.1.0.zip)
* [mortar list-detail layout](https://github.com/mozilla/mortar-list-detail): for list-based apps; comes with a layouts library and sets up a list-detail layout for you | [Download](/mortar-list-detail-0.1.1.zip)
* [mortar tab view](https://github.com/mozilla/mortar-tab-view): similar to mortar-list-detail but the initial layout has tabs at the button which let you switch between screens. | [Download](/mortar-tab-view-0.1.1.zip)
* [sutthisan](https://github.com/mozilla/mortar-layouts): for apps that need a simple server-side component | [Download](/sutthisan-master.zip)
* [Screencast](http://www.youtube.com/watch?v=nsvAwXBUll8) about mortar templates
* [mortar-layouts](https://github.com/mozilla/mortar-layouts): the UI library powering mortar list-detail and tab view | [README](/mortar-layouts-readme.markdown)
